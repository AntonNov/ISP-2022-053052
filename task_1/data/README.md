### 16. Что такое dev/null
dev/null — специальный файл в системах класса UNIX, представляющий собой так называемое «пустое устройство». Запись в него происходит успешно, независимо от объёма «записанной» информации. Чтение из /dev/null эквивалентно считыванию конца файла. В DOS есть псевдофайл NUL с аналогичными свойствами.
